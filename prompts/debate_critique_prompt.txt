## Debate Phase: Critique
## Your Role
Examine each OTHER agent's current answer and identify specific flaws in their reasoning.
Your own critique field must remain empty.

## Common Error Patterns to Target
- Omniscient fallacy: agent assigns knowledge to a character who was absent during the relevant event
- Belief-reality confusion: agent conflates what IS true with what the character BELIEVES is true
- Wrong inference order: agent applies 1st-order ToM when 2nd-order (or higher) is needed
- Information access error: agent ignores a character's absence at a key event
- Deception tracking failure: agent fails to propagate a false-belief update after a deceptive act
- If no clear flaw exists, state why you agree and confirm your answer

## INPUT (in user message)
- Your agent identity (e.g., "You are agent2")
- Scenario text and questions
- All agents' current reasoning (NOTE: final answers are hidden — critique logic only, not conclusions)

## TASK
For each OTHER agent:
1. Identify the most critical flaw in their reasoning using the error patterns above
2. Cite specific evidence: quote the exact story sentence or event number (e.g., "[Event 4]: 'Mary moved the box'")
3. Explain how the cited evidence contradicts their reasoning chain

Then state your own maintained or updated answer as a single choice letter.

## RULES
R1. Reference timeline events by explicit number (e.g., "[Event 2]", "[Event 5]") AND quote the relevant story sentence
R2. Be specific: name character + proposition + event, not vague assertions
R3. Do NOT infer or guess other agents' hidden final answers
R4. To change your own answer, you MUST cite which story evidence changed your reasoning and how
R5. Set your own agent's critique field to "" (empty string)
R6. my_answer — ONLY the single choice letter (e.g., "K"). No period, no text, no explanation.
R7. Output ONE valid JSON object. No markdown, no prose outside the JSON.

## OUTPUT FORMAT
{
  "critique_of_agent1": "<critique with event citations, or empty string if you ARE agent1>",
  "critique_of_agent2": "<critique with event citations, or empty string if you ARE agent2>",
  "critique_of_agent3": "<critique with event citations, or empty string if you ARE agent3>",
  "my_answer": "<single letter>"
}
