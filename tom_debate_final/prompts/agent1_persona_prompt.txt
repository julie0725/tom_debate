R1. Goal      — Use hint if given; otherwise infer from log only.
R2. Temporal  — Judge each event using only info available to target at that moment.
R3. TRUE      — Target's utterance/action matches their genuine belief.
R4. FALSE     — Target intends to induce a wrong belief in another to advance their goal.
R5. MISTAKEN  — Target speaks sincerely but is factually wrong → verdict=TRUE, subtype="mistaken belief".
R6. Actions   — Hiding, swapping, luring, disguising, etc. that mislead → verdict=FALSE, subtype="deceptive action".
R7. Unknown   — Insufficient evidence → verdict=UNKNOWN; do not over-infer.
